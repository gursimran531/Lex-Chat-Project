# 🤖 Amazon Lex Projects Collection

Welcome to the Amazon Lex Projects!  
This repository features step-by-step guides and implementations showcasing how to build intelligent, conversational chatbots using AWS Lex, along with advanced functionalities like custom slots, Lambda integration, and context carryover.

---

## 💡 Project Overview

This suite demonstrates the journey from a **basic Lex chatbot** to powerful, context-aware virtual assistants by walking through four project parts:

---

### 🗨️ Build a Chatbot with Amazon Lex (Part 1)
- Focus: Natural language understanding, intent creation, and robust fallback handling.
- Outcome: A simple yet functional AI chatbot using Amazon Lex.

---

### 🧩 Add Custom Slots to a Lex Chatbot (Part 2)
- Focus: Extending the chatbot to handle custom user input slots for specific data like balance inquiries.
- Outcome: Enhanced data collection & smarter intent handling.

---

### 🔄 Connect Amazon Lex with Lambda (Part 3)
- Focus: AWS Lambda integration for dynamic responses (e.g., providing random balance info).
- Outcome: Real-time, custom programmatic replies to user requests.

---

### 💾 Save User Info with a Lex Chatbot (Part 4)
- Focus: Using context carryover to remember user information across multiple intents.
- Outcome: Smooth, natural conversations and improved user experience.

---

## 🚀 Why This Project?
